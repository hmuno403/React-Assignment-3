CPRG 010 - Front-End web development using React.js
-----------------------------------------------------
This is in class demo for Module 4.

To run: 
	1- open the "express-web-4" folder in the command prompt and run the command "npm install" to install all project packages.
	2- run the webserver on the command prompt: npm run start 
	3- open the web brawser on: http://localhost:3000
